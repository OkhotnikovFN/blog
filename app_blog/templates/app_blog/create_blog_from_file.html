Создать блог из CSV файла

Файл должен сосотоять из двух колонок, разделителем является ':' (двоеточие),
Если поле содержит разделитель, его необходимо заключить в '"' (двойные кавычки)
Первая колонка - текст блога
Вторая колона - дата публикации в формате (год(4знака)-месяц(два знака)-число(два знака)T(латинская)часы(два знака):минуты(два знака))
Например (Текст блога:"2020-01-01T01:01"). Файл, должен быть в кодировке "UTF-8" или текст блога может неправильно интерпретироваться.
